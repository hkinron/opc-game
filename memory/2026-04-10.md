## Pixel Agents 项目

### MVP (v0.1) ✅
- 单文件 HTML, Canvas 2D, 零依赖
- Agent 状态机 (idle/walk/type/read/wait/error)
- BFS 寻路
- 像素风办公室 (desk, plant, couch, whiteboard, bookshelf, printer, coffee)
- 6 个 agent 角色 (Coder/Reviewer/Designer/Writer/Tester)
- 交互式 UI (点击切换状态, 悬浮详情, 状态栏)
- 自动模拟 (agent 自主切换工作状态)
- 可访问链接: OSS data URI (有时效性)
- UML 文档: `pixel-agents-mvp/docs/uml.md`

### Phase 1: TypeScript 工程化重构 ✅
- 拆成多模块 TypeScript 项目
- Vite 构建 (11 modules, 13.6KB / gzip 4.6KB)
- 目录结构:
  ```
  pixel-agents/
  ├── src/
  │   ├── types/index.ts      # 类型定义
  │   ├── engine/
  │   │   ├── TileMap.ts      # 地图
  │   │   ├── BFS.ts          # 寻路
  │   │   ├── Agent.ts        # Agent 状态机
  │   │   ├── Renderer.ts     # Canvas 渲染
  │   │   └── Game.ts         # 主循环
  │   └── main.ts             # 入口
  ├── index.html
  ├── package.json (pnpm)
  ├── tsconfig.json
  └── vite.config.ts
  ```
- pnpm 管理依赖 (npm 有 workspace 问题)
- 构建输出: `dist/`

### Phase 2: WebSocket 实时同步 ✅
- 后端: `pixel-agents/server/` — Node.js WebSocket 服务器 + JSONL 日志解析器
  - 监听 Claude Code JSONL 日志，解析 subagent_spawn/tool_use/error/waiting 事件
  - 广播到所有 WebSocket 客户端
  - 历史缓存 100 条事件，自动重连
- 前端: `src/engine/AgentWebSocket.ts` — WebSocket 客户端
  - 自动重连，状态同步
  - Game.ts 支持实时/模拟模式切换
  - URL 参数 `?ws=ws://host:8787` 启用实时模式
  - 状态栏显示 [LIVE] / [SIM]
  - Speech bubble 显示 agent 当前活动（文件名、命令等）
- 构建: 12 模块 → 16.7KB (gzip 5.7KB)

### Phase 3: Kanban 任务板系统 ✅
- **新文件**: `src/engine/KanbanBoard.ts` — 任务管理 + 序列化
- **更新模块**:
  - `types/index.ts` — 新增 Task, TaskState, Kanban 类型 + AgentState.FetchingTask
  - `engine/Agent.ts` — 任务工作流状态机 (8 个子状态), currentTask 字段, startFetchTask/startCompleteTask
  - `engine/TileMap.ts` — TileType.Kanban, KANBAN_BOARD 常量 (4 格宽, 位于上方墙壁)
  - `engine/Renderer.ts` — drawKanbanTile + drawKanbanOverlay, 任务卡片渲染, 优先级颜色
  - `engine/Game.ts` — autoAssignTasks(), 自动分配 + 点击 agent 分配, 状态栏显示进度
  - `engine/AgentWebSocket.ts` — task_change 事件类型, sendEvent/send 方法
- **工作流**: Idle → 走到Kanban → 取任务 → 回桌工作 → 走到Kanban → 标记Done → 回桌 → Idle
- **构建**: 13 模块 → 25.0KB (gzip 8.2KB)

### Phase 4: 视觉升级 ✅
- **ParticleSystem** (`src/engine/ParticleSystem.ts`):
  - 5 种粒子: spark/float/burst/steam/code
  - 物理模拟: 重力/上浮/减速
  - Code 粒子: 代码字符 (0,1,{,},</>,;等)
- **SpriteRenderer** (`src/engine/SpriteRenderer.ts`):
  - 16x20 程序化像素精灵
  - 5 角色专属外观 (眼镜/放大镜/调色板/钢笔/虫子)
  - 动态动画: 行走摆腿, 打字手臂, 状态指示器
- **环境效果**:
  - 打字: 代码字符+火花
  - 行走: 灰尘
  - 错误: 红色爆炸
  - 取任务: 金色闪光
  - 咖啡机: 蒸汽
- 构建: 15 模块 → 32.1KB (gzip 10.6KB)
