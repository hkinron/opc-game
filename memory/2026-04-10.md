## Pixel Agents 项目

### MVP (v0.1) ✅
- 单文件 HTML, Canvas 2D, 零依赖
- Agent 状态机 (idle/walk/type/read/wait/error)
- BFS 寻路
- 像素风办公室 (desk, plant, couch, whiteboard, bookshelf, printer, coffee)
- 6 个 agent 角色 (Coder/Reviewer/Designer/Writer/Tester)
- 交互式 UI (点击切换状态, 悬浮详情, 状态栏)
- 自动模拟 (agent 自主切换工作状态)
- 可访问链接: OSS data URI (有时效性)
- UML 文档: `pixel-agents-mvp/docs/uml.md`

### Phase 1: TypeScript 工程化重构 ✅
- 拆成多模块 TypeScript 项目
- Vite 构建 (11 modules, 13.6KB / gzip 4.6KB)
- 目录结构:
  ```
  pixel-agents/
  ├── src/
  │   ├── types/index.ts      # 类型定义
  │   ├── engine/
  │   │   ├── TileMap.ts      # 地图
  │   │   ├── BFS.ts          # 寻路
  │   │   ├── Agent.ts        # Agent 状态机
  │   │   ├── Renderer.ts     # Canvas 渲染
  │   │   └── Game.ts         # 主循环
  │   └── main.ts             # 入口
  ├── index.html
  ├── package.json (pnpm)
  ├── tsconfig.json
  └── vite.config.ts
  ```
- pnpm 管理依赖 (npm 有 workspace 问题)
- 构建输出: `dist/`

### Phase 2: WebSocket 实时同步 ✅
- 后端: `pixel-agents/server/` — Node.js WebSocket 服务器 + JSONL 日志解析器
  - 监听 Claude Code JSONL 日志，解析 subagent_spawn/tool_use/error/waiting 事件
  - 广播到所有 WebSocket 客户端
  - 历史缓存 100 条事件，自动重连
- 前端: `src/engine/AgentWebSocket.ts` — WebSocket 客户端
  - 自动重连，状态同步
  - Game.ts 支持实时/模拟模式切换
  - URL 参数 `?ws=ws://host:8787` 启用实时模式
  - 状态栏显示 [LIVE] / [SIM]
  - Speech bubble 显示 agent 当前活动（文件名、命令等）
- 构建: 12 模块 → 16.7KB (gzip 5.7KB)
